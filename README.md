# GBase Support Landing Page

GBase Support の商業施設向けAIコンシェルジュソリューションのランディングページです。

## 🌐 ページを見る

GitHub Pages で公開しています：

**https://lirhcoder.github.io/GBaseSupportSLP/**

## 📁 プロジェクト構成

```
GBaseSupportSLP/
├── index.html          # メインページ
└── assets/
    ├── faq.mp4         # FAQ機能デモ動画
    ├── realtime.mp4    # 音声対話デモ動画
    ├── shopmap-new.mp4 # フロアマップデモ動画
    ├── スマホ.png       # スマホUI画面
    └── 管理画面.png     # 管理画面スクリーンショット
```

## ✨ 機能

- 10言語対応のライブデモ（言語選択 → ウィジェット体験）
- 動画による機能紹介（音声対話・フロアマップ・FAQ）
- NEWoMan高輪 導入事例
- リード獲得フォーム（PDF資料ダウンロード）
- レスポンシブデザイン（モバイル対応）

## 🔧 GitHub Pages の設定方法

1. リポジトリの **Settings** → **Pages** を開く
2. **Source** で `Deploy from a branch` を選択
3. **Branch** で `main` / `/ (root)` を選択
4. **Save** をクリック
5. 数分後に上記URLでアクセス可能
